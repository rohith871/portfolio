# My Portfolio Website

This repository contains the source code for my personal portfolio website , showcasing my skills , projects and contact information . 
---

## Project Structure

```
portfolio/
├── index.html        — Home page
├── about.html        — About, education, skills
├── projects.html     — Projects
├── contact.html      — Contact form and details
└── style.css         — Single CSS file shared across all pages
```

---

## Pages

**Home** — intro section with name, role, short bio, skill chips and buttons to navigate to projects and contact.

**About** — education timeline from 10th grade to college and skills divided into frontend, languages and tools.

**Projects** — projects with images, descriptions, tech tags and links to GitHub and live demo.

**Contact** — contact details and a form to send a message directly.

---

## Features

- Flicker animation on the name using CSS keyframes
- Glass morphism cards with blur and semi-transparent background
- Fan card hover effect using CSS custom properties and transitions
- Sticky navbar with blur backdrop that stays fixed while scrolling
- Responsive layout with a hamburger menu using pure CSS — no JavaScript
- Hover effects on cards with glow, lift and image zoom
- Pulsing ring animation around the profile photo
- CSS variables for consistent theming across all pages
- Google Fonts — Orbitron for headings, Syne for body
